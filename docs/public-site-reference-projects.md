# Референс-проекты для публичного сайта

Публичный сайт не должен показывать бухгалтерские документы, счета, накладные и внутреннюю первичку как основной визуал. Эти вещи остаются сильной стороной ERP и могут упоминаться как контроль и порядок, но на сайте лучше продавать через понятные клиенту проекты, фото и сценарии работ.

## Правило подачи

- На сайте показываем дома, ремонты, отделку, фасады, кровлю, реконструкции и коммерческие помещения.
- Документы не публикуем как визуальные доказательства; вместо этого пишем коротко: `смета, фотоотчеты, акты и материалы под контролем`.
- Если фото не наше, показывать как `референс` или `пример`, не как выполненную работу.
- Для блока `Наши работы` использовать только реальные объекты компании или фото с разрешением заказчика.
- Для блока `Референсы` можно использовать AI/сток/подборки как направления будущих заявок.

## 15 направлений для карточек сайта

1. Одноэтажный современный дом
   - Для кого: семья, компактный участок, быстрый понятный проект.
   - Акцент: современный фасад, терраса, панорамные окна.

2. Одноэтажный дом из кирпича
   - Для кого: заказчики, которым важна надежность и классика.
   - Акцент: кирпич, теплая кровля, аккуратная входная группа.

3. Семейный коттедж
   - Для кого: загородный дом с террасой и большим общим пространством.
   - Акцент: уют, двор, семейный сценарий.

4. Двухэтажный современный дом
   - Для кого: участок ограничен, нужно больше площади.
   - Акцент: чистая геометрия, второй свет, балкон/терраса.

5. Двухэтажный дом из кирпича
   - Для кого: капитальный дом на долгий срок.
   - Акцент: кирпичный фасад, надежность, статус.

6. Двухэтажный дом с гаражом
   - Для кого: семья с автомобилями и хозяйственными зонами.
   - Акцент: гараж, навес, удобная планировка.

7. Компактный таунхаус
   - Для кого: городское жилье, инвестиционный или семейный формат.
   - Акцент: серия домов, компактность, современный фасад.

8. Реконструкция коттеджа
   - Для кого: старый дом, который нужно привести в порядок.
   - Акцент: усиление, обновление фасада, инженерия.

9. Ремонт и отделка фасада
   - Для кого: дом построен, но внешний вид и утепление нужно довести.
   - Акцент: леса, отделка, новый фасад.

10. Ремонт кровли
    - Для кого: замена/усиление крыши, мансарда, утепление.
    - Акцент: стропила, кровельное покрытие, безопасность.

11. Ремонт квартиры
    - Для кого: квартира под проживание или сдачу.
    - Акцент: чистая отделка, свет, напольные покрытия.

12. Отделка частного дома
    - Для кого: коробка готова, нужно довести дом до проживания.
    - Акцент: гостиная, дерево, свет, инженерия.

13. Ремонт ванной комнаты
    - Для кого: отдельный понятный ремонтный сценарий.
    - Акцент: плитка, сантехника, аккуратные узлы.

14. Ремонт кухни и гостиной
    - Для кого: семейная зона, кухня-гостиная, современная отделка.
    - Акцент: мебель, свет, удобство.

15. Ремонт офисных помещений
    - Для кого: коммерческие клиенты.
    - Акцент: сроки, рабочие места, инженерия, чистая приемка.

## Как использовать на сайте

- Первый экран: светлый позитивный строительный фон + калькулятор.
- Ниже: 15 направлений `Выберите тип проекта`.
- Внутри каждого направления показывать 10-15 карточек примеров.
- У каждой карточки: фото, короткое название, 2-3 преимущества, кнопка `Рассчитать`.
- После клика карточка подставляет тип работ в калькулятор и заявку.
- Реальные выполненные объекты отделить от референсов: `Наши работы` и `Идеи для проекта`.
