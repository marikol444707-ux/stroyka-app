## Общее правило для всех

Если действие связано с объектом, сначала выбрать правильный объект. Если система не показывает объект, значит доступ не назначен или пользователь вошел не под той ролью.

Нельзя создавать один и тот же факт дважды: выполненный объем идет из сметы/изменения к смете, ЖПР фиксирует факт, документы и деньги только отражают его.
